# Solar System Simulator

Three.jsを使用した3D太陽系シミュレーターです。楕円軌道の計算により、惑星や衛星の動きをリアルに再現します。

## 特徴

- **3D可視化**: Three.jsを使用した美しい3Dグラフィックス
- **正確な軌道計算**: ケプラー方程式を用いた楕円軌道のシミュレーション
- **ユリウス日ベース**: 正確な日時計算による惑星位置の算出
- **衛星のシミュレーション**: 月などの衛星も含む
- **時間の進行**: リアルタイムで時間を進めて惑星の動きを観察

## 含まれる天体

- 水星 (Mercury)
- 金星 (Venus)
- 地球 (Earth) - 月を含む
- 火星 (Mars)
- 木星 (Jupiter)
- 土星 (Saturn)
- 天王星 (Uranus)
- 海王星 (Neptune)
- 冥王星 (Pluto)
- ハレー彗星 (Halley's Comet)

## 技術スタック

- **Three.js**: 3Dグラフィックスライブラリ
- **Vite**: ビルドツール
- **julian-date**: ユリウス日の計算

## セットアップ

### 必要な環境

- Node.js (v14以上推奨)
- npm または yarn

### インストール

```bash
npm install
```

### 開発サーバーの起動

```bash
npm run dev
```

ブラウザで `http://localhost:5173` を開いてください。

### ビルド

```bash
npm run build
```

### プレビュー

```bash
npm run preview
```

## 使い方

1. 開発サーバーを起動すると、3D太陽系が表示されます
2. マウスでカメラを操作できます（OrbitControls）
3. 画面上部に現在の日時が表示されます
4. 時間は自動的に進行し、惑星が軌道を移動します
